# 新刊特設ページ（GitHub Pages用）

## このサイトの目的
高等学校数学の先生方に向けて，新刊『未来を拓く 高等学校数学 「探究的な学び」の授業づくり』の発売情報と概要を1ページで案内するための静的サイトです。

## GitHub Pagesでの公開方法
1. このリポジトリを GitHub に push します。
2. GitHub の **Settings > Pages** を開きます。
3. **Build and deployment** の Source で **Deploy from a branch** を選択します。
4. Branch は `main`（または公開したいブランチ） / Folder は `/ (root)` を選択し保存します。
5. 数分後に発行されるURLでページを確認します。

## 書影画像の差し替え方法
1. `assets/book-cover.jpg` を新しい書影画像に置き換えます。
2. ファイル名を変更しない場合，`index.html` の修正は不要です。
3. ファイル名を変更する場合は，`index.html` 内の `img src`，`og:image`，`twitter:image` も同時に更新してください。

## 購入リンクの変更方法
購入リンクは `index.html` の以下の箇所にあります。
- Hero セクションの「明治図書で詳しく見る」ボタン
- 「購入・詳細」セクションのボタン
- SNS紹介文面（`textarea` 内）

変更時はURLを一括で置換すると更新漏れを防げます。

## 本文を編集する場所
本文は主に `index.html` にあります。編集しやすいように，主要ブロックにコメントを付けています。
- Hero本文
- 「この本について」
- 「本書のメッセージ」
- 「SNSで紹介するための文面」
- 「著者プロフィール」

デザインを調整する場合は `styles.css` を編集してください。コピー機能などの動作は `script.js` を編集してください。
